# Custom Product Pages Configuration

Configure these 3 Custom Product Pages in App Store Connect under
App Store > Custom Product Pages. Each CPP reuses the same 6 default
screenshots in a different order with unique promotional text and
subtitle override.

## Default Screenshot Set (referenced by order number)

| # | Screen | File Prefix |
|---|--------|-------------|
| 1 | Home Dashboard | 01_home_dashboard |
| 2 | Habits & Streaks | 02_streaks_habits |

| 4 | Social & Leaderboard | 04_social_leaderboard |
| 5 | Achievements & XP | 05_achievements_xp |

## Notes
